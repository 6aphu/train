# TRAIN - Тренжерный зал

## Назначение проекта
Веб-приложение тренажерного зала "TRAIN". Позволяет пользователям просматривать расписание тренировок, знакомиться с тренерским составом, выбирать абонементы, а также включает систему регистрации и авторизации с сохранением данных в SQLite.

## Технологии
* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* База данных: SQLite

## Необходимые зависимости
Для работы проекта на компьютере должен быть установлен Node.js.
Библиотеки:
- express
- cors
- sqlite3

## Команды запуска
1. Откройте терминал в папке backend.
2. Установите зависимости командой: npm install
3. Запустите сервер командой: node app.js
4. Откройте файл index.html (из папки frontend) в любом браузере.
